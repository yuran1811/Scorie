# Installation

- Clone this repo via command

```bash
git clone https://github.com/yuran1811/Scorie.git
cd scorie
```

- env config

```bash
cp .env.example .env
```

then fill all the fields in `.env`

- Init and run the project

```bash
yarn
yarn dev
```

- Build and preview the project

```bash
yarn build
yarn preview
```

- Finish!
